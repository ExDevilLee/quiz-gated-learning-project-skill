# Agent Notes

This is a public skill repository. Keep agent-facing rules public-safe and portable.

## README Language Convention

- `README.md` is the default Chinese entry.
- `README-EN.md` is the English entry.
- Other language entries use `README-xx.md`, for example `README-JA.md` or `README-FR.md`.

## Synchronization Rule

When adding or modifying public-facing content, update `README.md` and all existing `README-xx.md` files so their meaning stays aligned.

Do not commit private memory, local-only paths, tokens, cookies, API keys, or unpublished personal/project details.
